## Projeto Portfólio em React TS

Este é um projeto portfólio desenvolvido em React com TypeScript, styled components, utilizando as bibliotecas Three.js para criação de modelos 3D e Framer Motion para animações.

### Pré-requisitos

- Node.js (v14.15.1 ou superior)
- Yarn ou NPM

### Instalação

1. Faça o clone deste repositório em sua máquina local:

git clone https://github.com/EduardoMG12/3dPortfolio.git

2. Acesse o diretório do projeto:

cd 3dPortfolio

3. Instale as dependências do projeto:

yarn install ou npm install

4. Execute o projeto:

yarn dev ou npm run dev

5. Abra o projeto no seu navegador acessando http://localhost:3000

### Ferramentas utilizadas

- React
- TypeScript
- Three.js
- Framer Motion
